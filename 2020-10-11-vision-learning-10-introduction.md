# Git 教程
## 一、什么是git

### git是一个版本控制系统
- 多版本分支
- 多人合作
- 简单低成本的备份及版本回退
- ……
	
## 二、安装git
	
### 在Ubuntu上安装git
打开终端，在命令行输入以下命令即可完成安装

	sudo apt-get install git

### 在Windows上安装git
	
在Windows上使用Git，可以从Git官网直接下载安装程序(https://git-scm.com/downloads)
，然后按默认选项安装即可，此处不做赘述。

安装完成后，在开始菜单里找到“Git”->“Git Bash”，弹出类似下图所示的类命令行窗口
